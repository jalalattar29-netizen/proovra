# Sicherheitsrichtlinie (Proovra)
**Gültig ab:** 2026-02-04  
**Kontakt:** security@proovra.com

## Meldung von Sicherheitslücken
Wenn Sie glauben, eine Sicherheitslücke gefunden zu haben, schreiben Sie an:
security@proovra.com

Bitte fügen Sie hinzu:
- eine Beschreibung des Problems
- Schritte zur Reproduktion
- Einschätzung der Auswirkung
- relevante Logs/Screenshots

## Koordinierte Offenlegung
Bitte veröffentlichen Sie Sicherheitslücken nicht, bevor wir eine angemessene Gelegenheit zur Untersuchung und Behebung hatten.

## Unser Versprechen
Wir nehmen Sicherheit ernst und reagieren so schnell wie praktisch möglich.
