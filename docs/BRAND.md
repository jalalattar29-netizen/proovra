# Proovra Brand Guide

## Logo
- Primary logo: `apps/web/public/brand/logo.svg`
- Icon usage: use the hexagon mark alone for app icon and favicon.

### Clear Space
Maintain clear space equal to the height of the inner checkmark around all sides.

### Minimum Size
- Digital: 24px height minimum.
- Print: 8mm height minimum.

## Colors
Primary Navy: `#0B1F53`  
Light BG: `#F8FAFC`  
Text Dark: `#1F1827`  
Teal: `#111927`  
Blue Accent: `#2F7DAA`  
Green Valid: `#1F9955`  
Red Error: `#C53030`

## Typography
- Inter (Latin)
- Noto Sans Arabic (Arabic)

## Usage Examples
- App icons: `apps/mobile/assets/icon.png`, `apps/mobile/assets/adaptive-icon.png`
- Web favicon: `apps/web/public/brand/favicon.ico`
- Web header logo: `apps/web/public/brand/logo.svg`
